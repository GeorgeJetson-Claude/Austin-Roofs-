# Austin Roof Quotes

Free roof quote comparison landing page for Austin, TX homeowners (2026 pricing guide).

**Live site:** https://austin-roofs.vercel.app

### Quick Overview
- Lead-gen page with form, 2026 Austin roofing costs table, trust badges, service areas, testimonials & FAQ.
- Mobile-responsive, built with Tailwind CSS + Font Awesome.
- Deployed on Vercel (auto-deploys from GitHub).

### Tech Stack
- HTML
- Tailwind CSS (CDN)
- Font Awesome icons
- Form backend: Formspree (or similar)
